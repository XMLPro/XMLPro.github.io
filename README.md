# 記事の書き方

ブログ記事、XMLProに関するニュース、その他投稿の3種類の記事をマークダウンで書くことが出来ます。

### ブログ

```bash
./new.sh blogs  
```

### XMLProに関するニュース

```bash
./new.sh news
```

### その他の投稿

```bash
./new.sh posts
```
